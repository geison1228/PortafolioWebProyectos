MIS PROYECTOS

Blackjack

Este proyecto consiste en un juego de Blackjack desarrollado en JavaScript, utilizando HTML para la estructura visual y CSS para la presentación de las cartas y botones. Se creó un sistema dinámico de baraja (deck) que genera todas las cartas combinando valores numéricos y figuras (A, J, Q, K). Posteriormente, el mazo se mezcla aleatoriamente utilizando la librería Underscore.js. El juego implementa funciones para tomar cartas, calcular valores y un sistema de mensajes que determina el ganador.

Oasis Tico

Descripción técnica: Este proyecto consiste en una página web enfocada en destinos turísticos de Costa Rica. El diseño visual fue implementado con CSS avanzado, incorporando Flexbox y Grid. Se aplicaron animaciones mediante @keyframes para generar efectos de movimiento. A nivel de interacción, se integró JavaScript para controlar elementos dinámicos, como un cursor personalizado con un ícono de perezoso, logrando un sitio totalmente responsive.
